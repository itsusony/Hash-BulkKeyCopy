# NAME

Hash::BulkKeyCopy - a xs module for clone hash with keys

# SYNOPSIS

    use Hash::BulkKeyCopy;

# DESCRIPTION

Hash::BulkKeyCopy is a xs module for clone hash by a key arr.
Double faster than PP source.
